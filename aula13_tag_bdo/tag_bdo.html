Aula 13 - tag bdo

A tag (Bidirectional Override) é usada para sobrescrever a direção do texto em um documento HTML, alterando o fluxo de leitura. Isso é útil em textos com idiomas que possuem direções de escrita diferentes, como o árabe (da direita para a esquerda) ou o inglês (da esquerda para a direita).

Para o nosso idioma o português, não tem muita utilidade essa tag, somente para idiomas que tem a escrita invertida.

Atributo próprio apenas 1: Dir - que define a direção do texto (ltr para esquerda para direita, ou rtl para direita para esquerda). O padrão do atributo dir é da esquerda para direita.
