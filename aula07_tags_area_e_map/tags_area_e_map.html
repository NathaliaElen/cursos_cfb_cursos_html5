Aula 07 - Tag area e map

Algumas tags dependem de outras: A tag de area é filha da tag map, ou seja, os elementos da tag area são sempre aninhados dentro de uma tag map.

A tag area define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis) através de definições de shapes sendo poly, circle or rect.

Quando falamos da tag map não estamos falando só de mapa, estamos mapeando áreas de clique.
