Aula 16 - tag button

A tag button em HTML é usada para criar um botão clicável na interface de uma página web. Ela pode executar ações como submeter formulários ou ativar scripts.

Pode ser totalmente formatavel via CSS.

Atributos comuns:

type: Define o tipo do botão.

disabled: Desativa o botão, tornando-o não clicável.

name e value: Associam um nome e valor ao botão, que são enviados ao servidor ao submeter um formulário.

autofocus: Foca automaticamente o botão quando a página é carregada.

Atributos relacionados ao form que podem ser aplicados no button:

form: Especifica a qual formulário o botão está associado. É útil quando o botão está fora do elemento form.

formaction: Define a URL onde os dados do formulário serão enviados ao clicar no botão. Esse atributo sobrescreve o action do próprio formulário.

formenctype: Define o tipo de codificação dos dados ao submeter o formulário. É usado principalmente com formulários que enviam arquivos.

formmethod: Define o método de envio do formulário (GET ou POST), sobrescrevendo o método definido no formulário.

formnovalidate: Desativa a validação de campos antes de enviar o formulário. Esse atributo é usado apenas em botões de tipo submit.

formtarget: Define onde abrir a resposta após o envio do formulário. Pode abrir em uma nova aba ou janela, por exemplo.
