Aula 15 - tag br

Quando usar o br

Quando você precisa de uma quebra de linha dentro de um texto sem criar um novo parágrafo.

Em situações onde a formatação do conteúdo exige quebras específicas, como endereços, poesia, letras de música ou em rótulos.

Quando não usar o br

Não deve ser usada para espaçamento entre blocos de conteúdo. Para isso, prefira usar CSS com margens e paddings.

Para criar novos parágrafos, utilize a tag p em vez de várias tags br.
