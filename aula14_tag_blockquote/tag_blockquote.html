Aula 14 - tag blockquote e tag cite

Tag Blockquote

A tag blockquote em HTML é utilizada para delimitar uma citação longa, geralmente separada do texto principal por indentação ou espaçamento adicional. Esse elemento é ideal para destacar trechos de textos retirados de outras fontes, proporcionando uma leitura mais clara e uma apresentação visualmente diferenciada. Ele também pode ser utilizado para incluir citações que se estendem por mais de uma linha, diferenciando-se da tag q, que é usada para citações curtas e inline.

Além de sua função de estilizar citações, o blockquote pode incluir o atributo cite, que é opcional, mas bastante útil. Esse atributo armazena a URL ou uma referência à fonte original da citação, facilitando a atribuição de créditos adequados e promovendo a transparência de informações. Por exemplo, se você incluir uma citação retirada de um artigo da web, pode adicionar a URL dessa página no atributo cite, indicando claramente de onde a informação foi obtida.

Tag Cite

A tag cite é usada para identificar o título de uma obra, como livros, artigos, filmes ou estudos, dentro de um documento HTML. Geralmente, o texto dentro da tag citeaparece em itálico, indicando que é uma referência à fonte ou título citado.

A tag cite não possui atributos próprios. Ela é usada principalmente para marcar o título de uma obra, mas não possui atributos dedicados como algumas outras tags. No entanto, pode ser combinada com atributos globais, como class, id, e style, para personalização e manipulação via CSS ou JavaScript.
