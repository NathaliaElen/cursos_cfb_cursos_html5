Aula 04 - Tag de hipertexto

Tag a é uma marcação de hipertexto/âncora, ou seja, links que podem nos direcionar para páginas internas ou externas.

Toda Tag a possuem marcação de abertura e fechamento.

Display: inline, todo conteúdo fica um ao lado do outro.

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

Um link não visitado está sublinhado e azul.
Um link visitado está sublinhado e roxo.
Um link ativo está sublinhado e vermelho.

Atributos/parâmetros

Vão servir para configurar a tag a.

Âncora que direcionar para o final da página

Âncora Vazia

Não posso considerar isso como um hipertexto, porque não tem o atributo href.

Meu perfil GitHub

Definição: Um link externo aponta para uma página ou recurso em um domínio diferente do que o usuário está atualmente.

Exemplo: Meu perfil GitHub Neste exemplo, o link leva o usuário ao site que tem meu perfil no GitHub, com um domínio diferente.

Uso Comum: Direcionar o usuário para recursos fora do site atual, como sites parceiros, artigos, ou fontes externas.

Diferencial Importante: Links externos geralmente incluem o protocolo (http:// ou https://), enquanto links internos normalmente utilizam caminhos relativos ou absolutos sem especificar o protocolo ou domínio.

Arquivo Aula 03- Tags de Títulos

Definição: Um link interno aponta para outra página ou recurso dentro do mesmo domínio (ou site).

Exemplo: Sobre Nós Neste exemplo, o link leva a uma página chamada "sobre.html" dentro do mesmo site.

Uso Comum: Navegação entre páginas de um site, links para seções da mesma página (usando âncoras), ou para arquivos hospedados no mesmo servidor.

Se estiver na mesma pasta, só indico o nome do arquivo. Senão vou ter que indicar o nome da pasta + /.

Arquivo Aula 03 dentro de outra pasta

Se estiver dentro de uma outra pasta, preciso indicar o nome da pasta e por uma barra. Exemplo: teste/

Para voltar na estrutura de um diretório que está na pasta anterior ao meu arquivo atual usamos o: ../
