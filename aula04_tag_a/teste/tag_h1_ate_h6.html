Aula 03 - Tags de Títulos de H1 a H6

Possuem uma hierarquia de títulos, sendo H1 o título mais relevante e H6 o título menos relevante, tem tag de abertura e fechamento. Não usamos essas tags para textos longos.

Display

block: fica um título abaixo do outro, o block sempre vai haver uma quebra de linha.

Elemento de fluxo

usado para marcar o conteúdo principal do corpo de uma página, elementos que eu visualizo no body.

Categoria

Fluxo e Cabeçalho.

Aceito em todos os navegadores

Chrome, Edge, Firefox, Safari e Opera.

Dom Interface

Uses HTMLElement.
